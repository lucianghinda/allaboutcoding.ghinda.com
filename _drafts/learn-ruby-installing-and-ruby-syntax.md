# Installing Ruby

I think the best way to install Ruby is to follow the instructions provided on the Ruby website: https://www.ruby-lang.org/en/documentation/installation/ 

## Linux or MacOS

If I were to choose what path to take from there I would choose [rbenv](https://github.com/rbenv/rbenv) or [asdf](https://github.com/asdf-vm/asdf) with [asdf ruby plugin](https://github.com/asdf-vm/asdf-ruby). 

<sup>[March 2022]</sup> I would recommend for learning purposes to install 3.0.3 as it is most probably that it is supported by most gems. After you will get familiar with the language you can move to the newest version. 

In case you need more help with installing rbenv or asdf here are some examples of tutorials with more detailed steps: 

- For any OS: https://gorails.com/setup
- Linux with rbenv: https://www.digitalocean.com/community/tutorials/how-to-install-ruby-on-rails-with-rbenv-on-ubuntu-20-04
- MacOS with rbenv: https://luther.io/macos/how-to-install-latest-ruby-on-a-mac

## Windows

I think the first choice for windows should be [RubyInstaller](https://rubyinstaller.org/).

One very good tutorial is the one from GoRails here: https://gorails.com/setup/windows/10

## What to do when you are stuck?

If you don't manage to install Ruby locally, for this part you can always use [Replit.com](https://replit.com)

# Learning Ruby Syntax

To have a quick start with Ruby you should try to read this resource: https://www.ruby-lang.org/en/documentation/ruby-from-other-languages

It explains Ruby syntax for people coming from other programming languages.
